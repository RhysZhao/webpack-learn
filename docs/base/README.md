# 前言

## 内容介绍

基础篇主要介绍 webpack5 的基本配置。

Webpack 配置多而杂，本章节主要是让你对 Webpack 有个基本的体验，能够进行简单配置。

Webpack 不难，只是杂而已。看完后，建议自己多尝试配置，熟能生巧。

## 你能学到些什么

1. 了解 webpack5 的核心概念，plugin/loader/assets 等
2. 能够对 webpack5 的基本配置
